Javascript之Cookie

什么是cookie

cookie是存储于访问者的计算机中的变量。每当同一台计算机通过浏览器请求某个页面时，就会发送这个cookie。

cookie是访问过的网站创建的文件，用于存储浏览信息。

在javascript中可以通过document.cookie来读取或设置这些信息。

cookie基础知识

  • cookie有大小限制，每个cookie所存放的数据不能超过4KB，如果超过，则返回空字符串
  • cookie是以文本形式存放在客户端的计算机中
  • cookie的格式：＝
  • cookie的有效期，默认的生命周期是浏览器关闭时候，如果希望在浏览器关闭后还能使用，则需要设置cookie的有效期
  • typeof document.cookie的结果是string
  • cookie有域和路径的概念，默认不能跨域访问cookie
  • 多个cookie可以存放在一个cookie文件中

设置cookie

在javascript中存取cookie时，必须使用Document对象的cookie属性：

function getCookie(name) {
var start = 0;
var end = 0;
if(document.cookie.length > 0) {
start = document.cookie.indexOf(name + ‘=’);
if(start !== -1) {
//获取cookie值的开始位置
start = start + name.length + 1;
end = document.cookie.indexOf(‘;’, start);
if(end === -1) {
end = document.cookie.length;
}
return decodeURI(document.cookie.substring(start, end));
}
}
return ‘’;
}
